# Zenith:
Zenith is a lifestyle management application designed to track physical energy, mental well-being, and productivity. It uses a centralized dashboard to provide insights into daily habits and long-term trends.
### Features
* **Energy Engine:** Log workouts and monitor calorie expenditure.
* **Vibe Tracker:** Record daily mood and mental frequency.
* **Velocity Tasker:** Manage productivity and task completion.
* **Lifestyle Ledger:** Track personal savings and financial growth.
* **Automated Reports:** Generates a full performance audit and sends it via email.

### Technical Stack
* **Language:** Python
* **Frontend:** Streamlit
* **Database:** Google Firebase (Firestore)
* **Authentication:** Firebase Auth
* **Analysis:** Pandas and Plotly
